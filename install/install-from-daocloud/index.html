使用Daocloud一键安装#

如果你没有VPS或云主机，可以选择Daocloud进行一键安装。实际上过程和使用docker-compose是一致的，只不过Daocloud提供了一些图形化的界面，和更快地构建方式。

官方演示站 http://mooder.daoaoo.io 是运行在Daocloud上，Daocloud是一个基于Docker的Paas平台，提供免费和收费两个选项。官方演示站使用的“4x专业版”，每个月48元。

创建硬盘卷#

创建数据库容器#

点击左侧“服务集成”，选择Postgres，创建服务实例，填入名字就好了。

现在Docker下暂时只支持使用Postgres，如果你想使用其他数据库，可以使用“源码安装”的方式安装在自己的服务器上。

创建完成后，将会得到Postgres的地址、用户名和密码，记下来，后面填写到环境变量中。

创建应用#

然后打开 https://hub.daocloud.io/repos/550ddb55-c6c7-4c21-8812-de334b92e2ae ，这是最新镜像。

点击右侧的“快速部署”，填写应用名称、运行环境（建议选择2X，也就是256M内存），端口80，

服务绑定，选择刚才创建的数据库；Volume绑定，选择刚才创建的硬盘卷，路径填写/data。

自定义环境变量，填写如下一些环境变量（每个环境变量的意义，请参考前面的文档）：

启动命令填写 --url-alias /media /data，意思是将url中的/media请求映射到/data目录中，也就是映射到前面创建的硬盘卷中。

OK，创建完成后，就开启运行即可。
